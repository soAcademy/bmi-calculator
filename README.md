# BMI Calculator

## Formula

`BMI = weight (kg) / (height (m) ^ 2)`

| BMI            | Weight Status     |
|----------------|-------------------|
| Below 18.5     | UnderWeight       |
| 18.5-24.9      | Healthy           |
| 25.0-29.9      | Overweight        |
| 30.0-34.9      | Obesity Class I   |
| 35.0-39.9      | Obesity Class II  |
| 40.0 and Above | Obesity Class III |
